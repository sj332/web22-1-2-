3장 문서 구조 표현하기

  1. 테이블
    2. <table>요소
    <table>
    • 표를 만들어 주는 태그로 단순히 표를 만들 목적 이외에도 문장이나 이미지를 적절히 배치하기 위해서도 많이 쓰임
    • 행(row) => 열(column) 만큼 셀(cell)
    • 테이블의 구성 요소: <table>, <tr>, <th>, <td>
    <tr>테이블 안에서 하나의 행(row)
    <th>테이블의 머리글을 쓸 때 사용. 굵은 글씨체로 가운데 정렬이 기본값
    <td>각 셀의 데이터를 쓸 때 사용, 왼쪽 정렬이 기본값


  2. 셀 합치기
    3. rowspan, colspan 속성
    • <td> 요소의 rowspan 속성과 colspan 속성을 사용
    • rowspan 속성: 위아래 줄(rows)의 셀 병합
    • colspan 속성: 옆 칸(column)의 셀 병합


  3. 표 본문 요소
    4. 표의 제목 - <caption>요소
    표의 머리줄, 몸체, 꼬리줄 표현
    몸체만 스크롤하는 것이 가능
    <tbody> 여러 번 사용할 수 있음


  4. 글 본문 요소
글 본문 요소
요소 이름 설명
머리말<header> 요소머리말 영역으로 웹의 문서의 제목, 소개 등 문서 관련 설명을 표기하는 요소를 포함
탐색 메뉴<nav> 요소다른 웹 문서나 문서 내의 이동을 위한 내비게이션 링크를 표현할 때 사용
독립된 본문<article> 요소웹 문서에서 주요 내용을 가진 독립적인 본문을 나타낼 때 사용
문서내 섹션 그룹<section> 요소웹 문서 내에서 절 단위로 구분하거나 의미가 비슷한 그룹으로 문서를 구분하여 묶기 위해서 사용
부수 정보<aside> 요소본문의 내용과 구별되는 별개의 정보를 표현하기 위하여 사용 예) 블로그에서 최신 댓글목록이나 토막상식, 배너 및 광고 등
꼬리말<footer> 요소웹 문서의 꼬리말에 해당하는 저자정보, 저작권 정보, 이용조건, 관련 링크 들을 나타내기 위해 사용
